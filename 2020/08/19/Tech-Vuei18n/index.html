Vue i18n configuration

Posted on 2020-08-19 In Tech , front-end
Use i18n in vue for international (language switcher).
  1. Install i18n 注意是vue-i18n！不要装i18n！！！
  2. Create ‘i18n’ file, containing a VueI18n object. Here we set 3 attributes.
    1) locale: language property. If locale is changed, the language of the whole project will change accordingly.
    2) fallbackLocale: default language. If other languages are missing, this language is used.
    3) messages: the true language content. For the messages field, we use require.context function, which is used to read the files under a folder. It takes 3 parameters:
    1) String directory: the directory to read the files
    2) Boolean useSubdirectories: if we read the subfolders recursively.
    3) RegExp regExp: regular expression for matching the files.

Then require.context returns a function, which have 3 attributes.

1) Funtion resolve(request): returns the relative path of request to directory
2) Function keys(): returns all filenames that matches regExp as an array.
3) String id: id of the context.

Consequently, the code above takes all files under ‘./locales’ and has a valid filename(ends with .json). And create a dictionary with keys being the filenames and values being the data in the file.
