83-Remove Duplicates from Sorted List*

Easy

Problem Description:

Given a sorted linked list, delete all duplicates such that each element appear only once.

My Solution:

  • For each valid pair of current node and current.next, replace current.next with the next element if current.val equals to current.next.val, else turn to the next pair.

88-Merge Sorted Array

Easy

Problem Description:

Given two sorted integer arrays nums1 and nums2, merge nums2 into nums1 as one sorted array.

My Solution:

Use index to record the next ordered element, starting from m+n-1. Each time place an element from either nums1 or nums2 to the back of nums1. Copy all numbers if nums2 is left with some undone elements.

100-Same Tree

Easy

Problem Description:

Given two binary trees, write a function to check if they are the same or not.
Two binary trees are considered the same if they are structurally identical and the nodes have the same value.

My Solution:

Recursively check if two trees are the same.
Return false if either child is null, children values are not the same, or either child returns false when calling isSameTree(child.left, child,right).
Return true otherwise.
