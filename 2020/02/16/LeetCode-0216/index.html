112. Path Sum

Easy

Problem Description:

Given a binary tree and a sum, determine if the tree has a root-to-leaf path such that adding up all the values along the path equals the given sum.

My Solution:

Define a new functon for pre-order traversal.
For a leaf node, check if the current path sum equals to the target.
For non-leaf node, update current path sum and recursively calculate sum.

118. Pascal’s Triangle

Easy

Problem Description:

Given a non-negative integer numRows, generate the first numRows of Pascal’s triangle. In Pascal’s triangle, each number is the sum of the two numbers directly above it.

My Solution:

Two nested for loop:
For each row, define an empty list, add the very first and last element. Calculate intermediate elements following the rule: num[i][j] = num[i-1][j-1]+num[i-1][j]

119. Pascal’s Triangle II

Easy

Problem Description:

Given a non-negative index k where k ≤ 33, return the kth index row of the Pascal’s triangle.
Note that the row index starts from 0.

My Solution:

O(n) space.
Define an ArrayList, starting from size 1.
Inside for loop, expand the list by inserting 0 to head, modify element j by the rule: num[j]=num[j]+num[j+1]

136. Single Number

Easy

Problem Description:

Given a non-empty array of integers, every element appears twice except for one. Find that single one.

My Solution:

Define a list. Iterate over all elements, if exists in the list, remove it; otherwise, add it to the list. Return the only integer left.

Preferred Solution:

O(n) time + O(1) space
Use XOR: a^0=a, a^a=0
After calculating xor of all elements, the result will be the single element.
