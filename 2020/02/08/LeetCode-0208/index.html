101-Symmetric Tree

Easy

Problem Description:

Given a binary tree, check whether it is a mirror of itself (ie, symmetric around its center).

My Solution:

Use 2 list to store each element in left and right node in particular order.

  1. Deal with current node, return false if one of 2 children is null.
  2. Recursively visit each node in left child and right child. Visit left ahead of right when going with left child; visit right ahead of left when going with right child.
  3. Compare 2 lists, return false if there are unmatched elements.

104-Maximum Depth of Binary Tree

Easy

Problem Description:

Given a binary tree, find its maximum depth.
The maximum depth is the number of nodes along the longest path from the root node down to the farthest leaf node.
Note: A leaf is a node with no children.

My Solution:

  1. Return 0 if root is null.
  2. Define another function, recursively visit the tree, each time return the max value of maxDepth(left) and maxDepth(right) plus one.

107-Binary Tree Level Order Traversal II *

Easy

Problem Description:

Given a binary tree, return the bottom-up level order traversal of its nodes’ values. (ie, from left to right, level by level from leaf to root).

My Solution:

  1. Define a queue (as LinkedList), store root node in the queue.
  2. Perform Breadth First Search:
    Loop until the queue is empty.
    Each time pop nodes which were inserted in the previous step(use queue.size() to decide).
    When iterating each node, add it to the list of current depth. And add its chidren to the queue.

141-Linked List Cycle

Easy

Problem Description:

Given a linked list, determine if it has a cycle in it.

To represent a cycle in the given linked list, we use an integer pos which represents the position (0-indexed) in the linked list where tail connects to. If pos is -1, then there is no cycle in the linked list.

My Solution:

Define a list.
Traverse each element, check if it already exists in the list.
If so, return true. Otherwise, add it to the list and deal with the next listnode.
