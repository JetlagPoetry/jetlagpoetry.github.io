46. Permutations

Posted on 2020-05-27 Edited on 2020-06-04 In LeetCode , 1-100题

Medium

题目描述:

Given a collection of distinct integers, return all possible permutations.

思路:

  • 使用递归构建，函数传入整个nums数组与数字头部指针，每次运算n个元素的permutation是由后n-1个元素的permutation生成。
  • 例：已知由元素2和3生成的permutation有[2,3], [3,2]。 那么对于每个结果，将新的元素1插入到不同的位置。
    [2,3] -> [1,2,3], [2,1,3], [2,3,1]
    [3,2] -> [1,3,2], [3,1,2], [3,2,1]
