42. Trapping Rain Water

Posted on 2020-06-12 In LeetCode , 1-100题
2pointer

Hard

题目描述:

Given n non-negative integers representing an elevation map where the width of each bar is 1, compute how much water it is able to trap after raining.

思路

  • 2 pointer求解
  • 分别记录两侧的指针l,r和两侧迄今为止的最高值lm,rm。由于积水的数量由这个小坑两侧高度的较小值决定，因此从两段同时向内侧进行，不断丢弃较小的那一侧。这样一旦保证了“留下来”的指针l和r都比曾丢弃的lm,rm要大，那么形成的积水数量是由lm和rm决定的。因此在指针向中心移动的过程中，就可以一个元素一个元素的添加面积值了，应当是lm-lcurrent或者rm-rcurrent。
  • 不断循环：
    比较l与r点的值，处理较小的那一侧，例如l：
    如果lm小于height[l]->那么更新lm的值。
    否则->将l这一列上的积水lm-l添加至答案的变量上。
    将左侧的指针向右移动。
    右侧同理。
  • 也可以使用动态规划、压栈法。
