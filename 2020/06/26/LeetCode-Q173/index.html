173. Binary Search Tree Iterator

Posted on 2020-06-26 Edited on 2020-06-27 In LeetCode , 101-200题
BST

Medium

题目描述:

Implement an iterator over a binary search tree (BST). Your iterator will be initialized with the root node of a BST.

Calling next() will return the next smallest number in the BST.

思路

  • 使用栈来实现迭代器，与中序遍历的实质是一样的：每个元素都会按照一定次序被压入栈中。下面依次来实现三个函数：
  1. 构造函数BSTIterator(TreeNode root)
    进行初始化，将根节点压入栈中，然后将最左一侧的所有节点压入栈中，直到某个节点没有左孩子。可以很轻易地知道，这时栈顶的元素是整个BST的最小值。
  2. 迭代函数next()
    每次应该拿取的元素，都是栈顶的第一个节点值。 所以，在拿取元素7之后，要判断它是否有右子树。如果有，就要把右孩子压栈，接下来进行右子树的处理了。
    之后，与构造函数的思路相同，在一个子树的根节点入栈之后，要把它最左侧的一系列孩子压栈，这样栈顶才能得到大于7的最小数8。
  3. hasNext()
    • 当栈空时表示所有元素都被遍历过，返回false，否则返回true。
    • 还有一种情况是，根节点是空的，也会被构造入栈，这时栈里有元素但不能返回true。（这个特殊情况也可以在构造函数中处理，在构造时不把空节点压栈就好）
