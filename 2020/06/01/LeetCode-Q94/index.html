94. Binary Tree Inorder Traversal

Posted on 2020-06-01 Edited on 2020-06-03 In LeetCode , 1-100题

Medium

Tree Traversal

(a) Inorder (Left, Root, Right) - 中序
(b) Preorder (Root, Left, Right) - 前序
(c) Postorder (Left, Right, Root) - 后序

题目描述:

Given a binary tree, return the inorder traversal of its nodes’ values.

思路一:

  • 简单递归实现中序遍历
  • 在每个节点，将左节点返回结果、当前节点的值、右节点返回结果依此添加到当前集合中并返回。
  • 时间复杂度：平均情况下，O(h)=(logN)；最坏情况下，O(h)=O(n)。
  • 空间复杂度：O(h)

代码一

思路二:

  • 使用循环和栈来模拟递归过程
  • 初始化空栈，当前节点设置为根节点。 每次循环时做两件事：一，不断地将当前节点最左侧的一系列子节点推入战中，直到没有左子节点。 二，步骤一做完后，弹出栈中最顶元素（想象为当前子树的左侧最远孩子），处理这个最顶元素（本题中是将节点值加入list），之后将右子节点设置为当前节点。当节点全部遍历完且栈为空时停止循环。
  • 时间复杂度：平均情况下，O(h)=(logN)；最坏情况下，O(h)=O(n)。
  • 空间复杂度：O(h)
