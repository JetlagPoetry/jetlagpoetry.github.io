930-Binary Subarrays With Sum*

Medium

Problem Description:

In an array A of 0s and 1s, how many non-empty subarrays have sum S?

Requirements:

Time: O(n)

My Solution:

Time Limit Exceed:: Tried with a O(n^2) method, starting with each index, count all possible subarrays.

Preferred Solution:

  1. First, O(n) to calculate array P.
  2. Search for how many pair of P[i] and P[j] satisfies P[i]+S=P[j].
    Use one for-loop to calculate P[j] and add the value of map.P[j] by 1.
    Each time get map.P[i] as the count mapped to this value as P[j].

575-Distribute Candies

Easy

Problem Description:

Given an integer array with even length, where different numbers in this array represent different kinds of candies. Each number means one candy of the corresponding kind. You need to distribute these candies equally in number to brother and sister. Return the maximum number of kinds of candies the sister could gain.

My Solution:

Use set to count number of unrepeated values.
Return the minimium between the value and half of the candies.
