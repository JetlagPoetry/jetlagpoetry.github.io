14-Longest Common Prefix

Easy

Problem Description:

Write a function to find the longest common prefix string amongst an array of strings.
If there is no common prefix, return an empty string “”.

My Solution:

  • 2-layer nested for loop.
    Iterate for each index in string, update prefix string if the character in all strings is the same, return otherwise. Also return if the end of a string is reached.

1122-Relative Sort Array

Easy

Problem Description:

Given two arrays arr1 and arr2, the elements of arr2 are distinct, and all elements in arr2 are also in arr1.
Sort the elements of arr1 such that the relative ordering of items in arr1 are the same as in arr2. Elements that don’t appear in arr2 should be placed at the end of arr1 in ascending order.

My Solution:

  • 2-layer nested for loop.
    For each element a2 in arr2, find all elements in arr1 that has the same value with a2, and swap them to the next correct position.
  • 2-layer nested for loop.
    After the last operation, all elements in arr that appear in arr2 is moved to the right place, leaving behind elements that do not show in arr2 in the rear of array. So we use another nested loop to sort the rest of numbers.
