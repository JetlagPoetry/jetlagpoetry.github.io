2. Add Two Numbers

Posted on 2020-05-27 Edited on 2020-06-05 In LeetCode , 1-100题

Medium

题目描述:

You are given two non-empty linked lists representing two non-negative integers. The digits are stored in reverse order and each of their nodes contain a single digit. Add the two numbers and return it as a linked list.

You may assume the two numbers do not contain any leading zero, except the number 0 itself.

思路:

  • 基础链表题目。
  • 使用两个指针分别指向l1和l2的节点，从头部开始。 循环相加每个pair，用布尔值记录进位信息。
  • 当其中一个链表被遍历完时，再循环加另一个非空的链表，同样记录每一步的进位信息。

3. Longest Substring Without Repeating Characters

Posted on 2020-05-27 Edited on 2020-06-03 In LeetCode , 1-100题

Medium

题目描述:

Given a string, find the length of the longest substring without repeating characters.

O(n) time: Use a string(hashset is better) to store current non-repeated substring. If current char is contained in the substring indexed at last, slide the left bound to last+1. Otherwise, increase the right bound and add current char to the string/set. Return the max value.

思路:

  • 使用一个字符串str来存储当前的无重复字符(推荐使用hashset)。循环遍历每个字符，如果当前字符在str中存在且位置为last，则重置字符串起始于last+1；否则，将当前字符添加到str末尾并记录当前字符串长度，返回最大值。
  • 时间复杂度：O(n)

11. Container With Most Water

Posted on 2020-05-27 Edited on 2020-06-04 In LeetCode , 1-100题

Medium

题目描述:

Given n non-negative integers a1, a2, …, an , where each represents a point at coordinate (i, ai). n vertical lines are drawn such that the two endpoints of line i is at (i, ai) and (i, 0). Find two lines, which together with x-axis forms a container, such that the container contains the most water.

思路:

  • 使用两个指针分别指向头和尾。 将两个指针指向的值中较小的那一个向中间移动，每一次记录area值，取最大返回。
  • 在每次计算中，对于较短的那一个值来说，当前面积已经是它所能得到的最大的面积，因此可以丢弃。

46. Permutations

Posted on 2020-05-27 Edited on 2020-06-04 In LeetCode , 1-100题

Medium

题目描述:

Given a collection of distinct integers, return all possible permutations.

思路:

  • 使用递归构建，函数传入整个nums数组与数字头部指针，每次运算n个元素的permutation是由后n-1个元素的permutation生成。
  • 例：已知由元素2和3生成的permutation有[2,3], [3,2]。 那么对于每个结果，将新的元素1插入到不同的位置。
    [2,3] -> [1,2,3], [2,1,3], [2,3,1]
    [3,2] -> [1,3,2], [3,1,2], [3,2,1]
