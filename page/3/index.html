13. Roman to Integer

Posted on 2020-01-31 Edited on 2020-06-04 In LeetCode , 1-100题

Easy

题目描述:

Roman numerals are represented by seven different symbols: I, V, X, L, C, D and M.

For example, two is written as II in Roman numeral, just two one’s added together. Twelve is written as, XII, which is simply X + II. The number twenty seven is written as XXVII, which is XX + V + II.

Roman numerals are usually written largest to smallest from left to right. However, the numeral for four is not IIII. Instead, the number four is written as IV. Because the one is before the five we subtract it making four. The same principle applies to the number nine, which is written as IX. There are six instances where subtraction is used:

  • I can be placed before V (5) and X (10) to make 4 and 9.
  • X can be placed before L (50) and C (100) to make 40 and 90.
  • C can be placed before D (500) and M (1000) to make 400 and 900.
    Given a roman numeral, convert it to an integer. Input is guaranteed to be within the range from 1 to 3999.

思路:

  • 首先把每个字符与它所对应的值存储在map中。
  • 遍历字符串中的每个字符，如果当前字符对应的值比后者大或相等，那么在结果上添加这个值；如果比后面的字符更小，在结果的基础上减去这个值。
  • XXVII等于X+X+V+I+I，同时IV等于-I+V。

26.Remove Duplicates from Sorted Array

Posted on 2020-01-31 Edited on 2020-06-04 In LeetCode , 1-100题

Easy

题目描述:

Given a sorted array nums, remove the duplicates in-place such that each element appear only once and return the new length.
Do not allocate extra space for another array, you must do this by modifying the input array in-place with O(1) extra memory.

思路:

  • 使用下标记录至今为止的无重复数组个数。
  • 遍历每个元素，如果当前元素与上一个处于无重复数组中的元素相同，则跳过；不相同，那么将无重复数组区间的下一个设置为当前元素，并将记录的下标右移一位。
  • [0,1,3,3,3,4,5,5,6]，当前正确下标为2，那么检测第3、4个元素发现都是3，已经在选中区间[0,2]中出现过。检测到第五个元素时发现没有出现过，那么设置：
  • [0,1,3,4,3,4,5,5,6]，并赋值当前正确的下标至3。

31. Next Permutation

Posted on 2020-01-29 Edited on 2020-06-04 In LeetCode , 1-100题

Medium

题目描述:

Implement next permutation, which rearranges numbers into the lexicographically next greater permutation of numbers.
If such arrangement is not possible, it must rearrange it as the lowest possible order (ie, sorted in ascending order).
The replacement must be in-place and use only constant extra memory.
Here are some examples. Inputs are in the left-hand column and its corresponding outputs are in the right-hand column.

思路:

  1. 从数组末尾寻找第一组相邻的数字，符合 nums[i-1] 小于 _nums[i]_，即找到第一个需要更换的位置 _i-1_。
  2. 在 num[i:] 中寻找比它最大的数字中最小的一个，记录下标为min
  3. 对调 nums[i-1] 和 nums[min] 。
  4. 把 nums[i:] 按照递增的方式重新排序。

例：

  • [1,2,5,3,1] 循环找到第一个pair(2,5)，后面最小的符合要求的数字为3，也就是说要把3提前到2所在的位置。
  • [1,3,5,2,1] 将3之后的数字重新排序
  • [1,3,1,2,5] 得到结果。

36. Valid Sudoku

Posted on 2020-01-29 Edited on 2020-06-04 In LeetCode , 1-100题

Medium

题目描述:

Determine if a 9x9 Sudoku board is valid. Only the filled cells need to be validated according to the following rules:

  1. Each row must contain the digits 1-9 without repetition.
  2. Each column must contain the digits 1-9 without repetition.
  3. Each of the 9 3x3 sub-boxes of the grid must contain the digits 1-9 without repetition.
    The Sudoku board could be partially filled, where empty cells are filled with the character ‘.’.

思路:

简单的O(n^4)解法。（COMP208期末考试出过这道题，各位小朋友基本都会做。。哎）

  • O(n^2) 循环
    检查每行是否有重复元素
  • O(n^2) 循环
    检查每列是否有重复元素
  • O(n^4) 循环
    检查每个3x3方格是否有重复元素。 把所有字符添加到一个字符串中，并用 indexOf() 查找是否有重复。
