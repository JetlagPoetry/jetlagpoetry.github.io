6. ZigZag Conversion

Posted on 2020-06-04 Edited on 2020-06-05 In LeetCode , 1-100题

Medium

题目描述:

The string “PAYPALISHIRING” is written in a zigzag pattern on a given number of rows like this: (you may want to display this pattern in a fixed font for better legibility)

And then read line by line: “PAHNAPLSIIGYIR”

思路:

  • 画风清奇的一道题，可以当做找规律的题来做。
  • 对于总共n行的zigzag，不难发现每(2n-2)个元素为一个周期。
  • 假设当前为第j个周期，坐落在第i行的元素为：
    1. 当i为第0行或第n-1行时，只有一个字符，分别是j*(2n-2)和j*(2n-2)+n-1。
    2. 当i是中间行时，每个周期包含两个字符，分别是j*(2n-2)+i和(j+1)*(2n-2)-i。
  • 因此从第0行到第n-1行，通过循环依此添加该行所有的字符即可。
  • 需要注意的是，如果n=1，那么很难跳出循环，所以需要在函数首部进行额外的检查。
